# **App Name**: SafeRice Analyzer

## Core Features:

- Simulated Authentication: Simulate user authentication with local state handling for signup and login functionalities.
- Central Dashboard: A central dashboard with a welcome message and navigation to all application features.
- Live Image Capture: Live image capture functionality to be used in AI simulations.
- Rice Classification Simulation: AI tool simulation for rice classification (long grain, short grain, broken) based on live image.
- Plant Disease Detection: AI simulation to detect diseases (e.g., Bacterial Blight, Sheath Rot) based on the captured image. The tool will decide the diagnosis.
- Plant Health Check: Provide random plant health score based on image captures
- Display Additional Information: Display additional, pre-baked information (e.g. cuisine ideas, dummy medicines info) to the user based on simulated diagnoses.

## Style Guidelines:

- Primary color: HSL(50, 70%, 50%) which converts to a bright, hopeful gold, symbolizing rice (#B89B29)
- Background color: HSL(50, 15%, 95%) which converts to an off-white (#FDFDF4).
- Accent color: HSL(20, 70%, 40%) which converts to a rich burnt orange, to accent and call attention (#B85929).
- Headline font: 'Playfair' serif for titles; body font: 'PT Sans' sans-serif.
- Use minimalist icons related to rice and plant health, for intuitive navigation and visual appeal.
- Mobile-responsive layout using Tailwind CSS grid and flexbox, ensuring accessibility across different devices.
- Subtle loading animations and transitions for a smooth user experience, especially when simulating AI responses.
